C#

Trojan

Trojan was a collaborative project created by myself and six other RMIT students. I took leadership of the coding team for this project, and we worked to create an endless runner game. A large amount of the code base was written by me, with the visual elements being created by others in the group. The commit history for the project is viewable through the link below.

Prior to the project, I had no experience with C#, so I had to learn as I went along. The game features random enemy layouts, increasing difficulty and saves the players scores to file after every round.

The GitHub repository for Trojan can be found here.

A video demonstration, albeit without sound, can be found below.

MicroPython

Simple BBC micro:bit game

Using the included compass of the BBC micro:bit, I created a simple game in MicroPython. The game randomly chooses one of four directions, and the player must quickly turn their devices in that direction and press 'A'.

The .py file can be found here.
